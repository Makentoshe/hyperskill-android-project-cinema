Description

Running a cinema theatre is no easy business. To help our friends, let's add statistics to your program. The stats will show the current income, total income, the number of available seats, and the number of occupied seats.

Objectives

Add a simple layout where the main info about the cinema room will be displayed: 


- TextView with id cinema_room_total_income should have text as an example: "Total cinema income: 123.45$"

- TextView with id cinema_room_current_income should have text as an example: "Current cinema income: 0.00$"

- TextView with id cinema_room_available_seats should have text as an example: "Available seats: 56"

- TextView with id cinema_room_occupied_seats should have text as an example: "Occupied seats: 0"
