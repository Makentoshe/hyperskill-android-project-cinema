Description

Good job: our friends really liked your program! Now they want to expand their app by adding a wholesome feature, named "profit". This feature depends on movie duration, ratings, and available seats in the cinema room. This feature allows defining the best cinema session. Help them calculate the "profit" from the current session, and, maybe, let them raise the ticket cost for the best/popular movies.

Objectives

In this stage, you need to read two positive numbers from the activity intent property: the movie duration in full minutes (under the "DURATION" string key) and the movie rating in the range 0 until 5 (under the "RATING" string key). The ticket price is determined by the following rules:

  • The most comfortable movie duration is 90 minutes, so, the formula for predicting profit: \[y = -{1 \over 90} x^2 +2x + 90\]where x is a movie duration in full minutes, and y is a movie_duration_profit
  • The base formula for the ticket price is (rating * movie_duration_profit) / available_seats
  • If the rating doesn't declare from intent, the value should be equal to 4.5.
  • If the movie duration doesn't declare from intent, the value should be equal to 108.
  • The calculation result should look like "Estimated ticket price: your-value-here$" and placed in a TextView with id cinema_room_ticket_price.
